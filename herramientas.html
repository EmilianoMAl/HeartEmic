The Positive API - Documentación

API REST Gratuita

Accede a miles de frases motivacionales, afirmaciones y mensajes de bienestar a través de endpoints simples. Sin necesidad de autenticación para uso básico.

  • Sin límite de requests para uso personal
  • Respuestas en formato JSON
  • Múltiples categorías de contenido
  • Soporte CORS habilitado
  • Actualizaciones constantes de contenido
Explorar API Completa
Ejemplo de uso 
fetch('https://positive-api.online/api/random')
  .then(res => res.json())
  .then(data => console.log(data.quote));
